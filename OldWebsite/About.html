Lenguajes de programación

El lenguaje preferido y el que más utilizo es c++, pero también sé usar bastante bien c#, java y javascript. Actualmente estoy aprendiendo HLSL y python por mi cuenta en mi tiempo libre.



Entornos de desarrollo

El motor de videojuegos con el que más contacto he tenido es Unity, aunque también he trabajado con Unreal. A la hora de programar suelo usar tanto Visual Studio Code como Visual Studio, por lo que ya tengo experiencia en el manejo de proyectos y soluciones en ese entorno. Sé usar también OGRE, llegando a crear un motor de videojuegos en 3D usando esta herrramienta. Además de estas herramientas más convencionales, también he creado juegos usando Processing y Phaser.
