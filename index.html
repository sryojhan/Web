Acerca de la página

Esta página surgió de la necesidad de juntar todos mis proyectos en un único lugar. En vez de usar alguna herramienta externa decidí desarrollar esta página por cuenta propia para mejorar mis escasos conocimientos de desarrollo web. Finalmente, a la hora de hacer hacer la página decidí hacerla exclusivamente con html, javascript y css puro. Principalmente, porque quería entender bien el funcionamiento de las bases de una página web antes de aprender alguna biblioteca como Bootstrap. Esta página no es lo que se llamaría "responsive" vaya.

Para hacer la página más interesante visualmente decidí añadir algunos elementos dinámicos. Estos elementos fueron hechos con un canvas y javascript. Dichos elementos son, el visualizador de constelaciones y el tetris automático. Destacar entre estos dos el tetris, con un mini algoritmo que coloca cada pieza en la mejor solución posible sin contar giros, aunque no es del todo perfecto, pero bueno, nadie lo es.
