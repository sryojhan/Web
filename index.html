Acerca de la página

Esta página ha sido completamente desarrollada por mí usando html, javascript y css básico. Soy plenamente consciente de que esté acercamiento a la hora de hacer una página es bastante primitivo pero al ser la primera página web que hago quería dominar primero las bases antes que aprender Bootstrap por ejemplo.

Esta página surgió de la necesidad de juntar todos mis proyectos en un único lugar. Nunca habíoa desarrollado una página web 'seria' antes así que esta era una oportunidad perfecta para poner a prueba y mejorar mis escasos conocimientos de desarrollo web. Sé que la página podría haber resultado mucho mejor si la hubiera hecho de otra manera ( esta página ahora mismo no es lo que se dice 'responsive' ), pero estoy bastante contento tanto con las distintas cosas que he aprendido como con el resultado final.

Una de las cosas que tenía claro que quería añadirle a la página web era el crear elementos visuales de forma procedural usando javascript. En mi opinión dichos elementos dotan de dinamismo a la página y creo que queda visualmente todo más bonito. De entre estos elementos, quiero destacar el tetris de la sección de proyectos, pues aunque me costó mucho hacer que quedase exactamente como quería usando css, el resultado final es bastante cercano a mi idea inicial, aunque el algoritmo de colocación de piezas no sea el mejor creado, al menos hace lo que puede.
