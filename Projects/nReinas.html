nREINAS

Con Love tendás la posibilidad de crear una gran variedad de juegos en 3D.


Love Engine es un motor realizado por nueve personas. Está construido con la capacidad de renderizar elementos tanto en 2D y 3D, cuenta con un motor de físicas hecho implementando Bullet. Se usa SDL para el Input tanto de ratón y teclado como mando. Para el manejo de sonido se utiliza Fmod y por último, las escenas se cargan como datos utilizando scripting hecho en Lua

Ver el código en GitHub
